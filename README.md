# Java Basics – Gustavo Bauer

Este repositorio recopila una colección organizada de ejercicios fundamentales de programación en **Java**, desarrollados durante mis estudios de DAM.  
Incluye múltiples prácticas completas agrupadas por temas, con el objetivo de demostrar una base sólida en los conceptos esenciales del lenguaje.

---

## Contenido del repositorio

Los ejercicios están organizados por carpetas temáticas, cada una relacionada con un área concreta del aprendizaje:

### **castingyparsing/**
Ejercicios centrados en:
- Conversión de tipos (casting implícito y explícito)
- Parsing de datos desde cadenas
- Manejo de conversiones entre tipos primitivos y objetos

### **math/**
Ejercicios utilizando la clase **Math**:
- Métodos matemáticos básicos
- Cálculo de valores aleatorios
- Operaciones trigonométricas
- Simulación de dados y cálculos numéricos

### **scanner/**
Programas basados en entrada de datos por consola usando **Scanner**:
- Lectura de diferentes tipos de datos
- Validación de entrada del usuario
- Ejercicios interactivos

### **functions/**
Ejercicios enfocados en:
- Creación de métodos
- Parámetros y valores de retorno
- Reutilización de código
- Modularización

### **EJava**, **ebpptjava**, **pptjava**
Carpetas con prácticas variadas que incluyen:
- Estructuras de control (if, else, switch)
- Bucles (for, while, do-while)
- Arrays unidimensionales y multidimensionales
- Ejercicios de presentación y lógica básica
